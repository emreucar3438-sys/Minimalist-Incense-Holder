# Minimalist-Incense-Holder
A simple and modern 3D printable incense holder design, modeled in Fusion 360.
# Overview
This project features a minimalist incense holder designed for 3D printing. It is compact, easy to print, and functional for daily use.

# Files
Incense Holder.stl: The ready-to-print 3D model.

# Printing Recommendations
Material: PLA or PETG recommended.

Infill: 15-20% is sufficient.

Supports: Depending on your specific design, supports may not be needed, but please check your slicer preview.
